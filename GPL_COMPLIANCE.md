# GPLv3 Compliance (Binary Distribution)

This repository contains GPLv3-licensed software (`LICENSE.txt`).

When distributing APKs or other binaries built from this repository, you must comply with GPLv3 requirements, including providing Corresponding Source.

## Source Availability

For binaries distributed from this fork, Corresponding Source is made available from this repository:
- Repository: this GitHub repository
- License: GPLv3 (`LICENSE.txt`)
- Third-party notices: `THIRD_PARTY_NOTICES.md`

If you distribute binaries outside this repository (mirrors, direct APK hosting, device preload, etc.), you are responsible for ensuring recipients can obtain the corresponding source under GPLv3 terms.

## Recommended Release Checklist

For each binary release:
1. Publish source commit/tag for the exact binary.
2. Include links to:
   - `LICENSE.txt`
   - `GPL_COMPLIANCE.md`
   - `THIRD_PARTY_NOTICES.md`
3. Clearly mark release as modified fork/port where applicable.
